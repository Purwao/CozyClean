# Cozyclean API 🚀🧼

Welcome to the **Cozyclean API**! This project is built using **Laravel** and serves as our **Tugas Akhir (TA)** for the **2024-2025** academic year.

## Overview

The Cozyclean API serves as the backend of our Laundry Application. It implements all the necessary CRUD (Create, Read, Update, Delete) operations for the following entities:

- **Orders**
- **Order Details**
- **Services**
- **Types**
- **Users**

## Features

- **Order Management**: Create, retrieve, update, and delete orders effortlessly.
- **Order Details**: Manage detailed information about each order.
- **Services**: Easily add, update, or remove services offered by Cozyclean.
- **Type Management**: Categorizes into different types.
- **User Management**: Handle user registration, and profile management.
